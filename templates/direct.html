Notes

Country: select a country to begin.
Unit of Measure: select Metric Ton to see data displayed in tons, or Number of Fish to see only the negligible amounts of fish reported (they were reported this way rather than as fractional tons).
Year Range: select a year to change the data on the map.
Gear: select the type of fishing gear from the list that you'd like to see visualized.
Species: select a fish species to filter on a specific one. By default, you'll see all reported for that country and other filters you've selected.
